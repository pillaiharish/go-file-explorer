## On Mac
Make the script executable:
```bash
chmod +x run_server.sh
```
Run the script:
```bash
harish $ bash mac_run_server.sh 
2024/06/20 22:42:47 Server started at :8282
```

## On Windows
Edit the path variable in the script then run the script in the 
```bash
.\run_server.ps1
```
